### Entry Flow
1. User lands on the platform.
2. Immediate explanation of purpose, scope, and limitations.
3. Clear disclaimer separating guidance from advice.

---

### Core Flow
1. User selects income or asset category.
2. User inputs basic figures (income, turnover, asset type).
3. Platform displays:
   - Applicable tax band
   - Indicative tax impact
   - Relevant exemptions and reliefs
4. Compliance checklist is generated.
5. User downloads PDF summary.

---

### Chat Interaction Flow
- User asks predefined questions about tax rules.
- System responds using rule-based explanations only.
- No personalized advice or interpretation beyond rules.

---

### Exit Flow
- User leaves with:
  - Clarity
  - Downloaded reference material
  - No stored data
